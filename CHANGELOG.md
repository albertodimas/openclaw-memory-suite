# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-01
### Added
- Initial public release of OpenClaw Memory Suite.
- Multi-layer memory plugins (episodic, procedural, entity, graph, goal, timeline, sentiment, blackboard, toolskill, meta).
- Routing stats helper and meta-memory reporting.
- LTM reranker plugin for Ollama (Qwen3 Reranker).
- Example config and MIT license.
